# 会员注册小程序

这是一个基于微信云开发的会员注册小程序，用户可以注册成为会员并管理个人资料。

## 功能特性

- 微信授权登录
- 用户资料注册与编辑
- 头像上传
- 多种个人信息收集（如健康关注、爱好等）
- 基于云数据库的数据存储

## 云开发配置

### 1. 启用云开发

1. 在微信开发者工具中，点击"云开发"按钮
2. 如果是首次使用，根据提示创建云开发环境
3. 记录下环境ID，并在 `app.js` 中更新 `env` 参数

### 2. 上传并部署云函数

需要部署以下云函数：

- login：用于获取用户openid
- userOps：用于用户数据的增删改查

部署方法：
1. 在微信开发者工具中，右键点击cloudfunctions文件夹中的每个云函数
2. 选择"上传并部署：云端安装依赖"

### 3. 创建数据库集合

需要创建以下集合：

- users：存储用户信息

创建集合后，需要设置适当的权限，建议设置为：
- 所有用户可读
- 仅创建者可写

## 开发说明

### 项目结构

- cloudfunctions：云函数
  - login：登录云函数
  - userOps：用户数据操作云函数
- pages：小程序页面
  - auth：用户授权页面
  - register：用户注册/编辑页面
  - member：会员信息页面

### 开发环境要求

- 微信开发者工具最新版本
- 基础库版本 2.2.3 或以上

## 测试和部署

1. 在微信开发者工具中完成功能测试
2. 确保所有云函数已上传部署
3. 确保项目基础库版本兼容
4. 上传代码并提交审核

## 注意事项

- 首次使用时需要正确配置云开发环境ID
- 在手机预览时可能需要先开启调试模式
- 头像等图片文件存储在云存储中，注意控制文件大小和数量
